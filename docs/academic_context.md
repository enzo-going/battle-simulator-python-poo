# Contexto do Projeto

Este repositório é um fork de um projeto acadêmico desenvolvido para a disciplina de Programação Orientada a Objetos.

O projeto original implementa um simulador simples de batalha em Python, com bases, tropas e ataques por turno.

## Objetivo deste fork

Este fork será usado para refatoração, organização e evolução do projeto, com foco em:

- melhorar a estrutura do código;
- aplicar melhor os conceitos de Programação Orientada a Objetos;
- organizar os arquivos do projeto;
- melhorar a documentação;
- tornar o projeto mais apresentável como portfólio;
- futuramente adicionar novas funcionalidades.

## Estado atual

O código ainda mantém a estrutura original do trabalho acadêmico e será evoluído gradualmente.
